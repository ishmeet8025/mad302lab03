## Description
This Android application demonstrates:
- Runtime permission handling (Location)
- Asynchronous operations using Kotlin Coroutines
- Simulated API data fetching
- Error handling (network failure simulation)

## Features
- Fetch Data button
- Displays loading, success, and error states
- Handles permission denial gracefully
- Simulates network delay using delay(2000)

## Technologies Used
- Kotlin
- Android SDK
- Coroutines (lifecycleScope)

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Run on emulator or physical device
4. Grant location permission
5. Click "Fetch Data"
